Label Manager

Download text files from Livestream Alerts to include into your streaming software.

The Label Manager Chrome extension runs as a Chrome app, and it downloads files to a location you specify while running. Specifically, for each "Recent" list you have configured, the data will be downloaded as a text file to the directory you specify, formatted as type-id (as listed on the recents list.

You will need to have Chrome installed to use the Label Manager, but once it is installed you can create a shortcut on your desktop.
